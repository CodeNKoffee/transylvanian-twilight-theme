## üöÄ Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Transylvanian Twilight"
4. Click Install
5. Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
6. Type "Preferences: Color Theme"
7. Select "Transylvanian Twilight"
